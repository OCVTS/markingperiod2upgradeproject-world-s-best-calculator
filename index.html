World's Best Calculator

My project is a calculator, allowing the user to input their calculations and get the correct answer. The purpose is to make math easier for the users of this site and help reduce any stress that comes along with mathematical equations.

    How I created and upgraded my project:

  • HTML: The HTML was very easy, mainly just the standard stuff you'd expect. I linked the JS and CSS, added buttons that were a part of the calculator, put in the meta tags, and I added some numbers on the sides of the screen just for stylization.
  • CSS: I didn't really need to change much with my upgrade. I switched to a color scheme of Cafe Noir, Tiffany Blue, Ghost White, Licorice, Xanthous, and Steel Blue. I also changed up the size of my elements a bit so they could all fit onto one screen.
  • JavaScript: This was the tricky part, especially getting certain operators to work. The JS is where I put in multiple different functions to make the calculator work, like telling the computer which operations to perform when their buttons are pressed. It even enables the user to input multiple numbers and operators into one equation.
  • Challenges: I did face quite a few challenges, all while adding in the operators. I attempted to implement decimals and fractions but could not figure out how they work for the life of me, so I just kept them as is but added buttons for a random number and one for PI to compensate. It was also a bit of a struggle to implement the ability to have multiple operators and more than two numbers in a single equation, but I asked ChatGPT and it helped me figure out how to do what I needed.
  • Added Features: I didn't have to add much besides some new buttons. I added buttons for random numbers, fractions, decimals, Pi(Ï€), rounding, and also exponents.
  • My Toughts: While a bit frustating at times, I thought this project was pretty enjoyable. And I always do like throwing random colors together and seeing if they all look good with each other.
